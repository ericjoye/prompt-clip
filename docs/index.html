FAQ

Does it work with ChatGPT, Claude, and Gemini?

Yes. Prompt Clip copies a formatted prompt to your clipboard — paste it into any AI tool.

Does it collect my data?

No. Zero network requests. Everything runs locally. See our privacy policy.

Is it really free?

The core extension with 5 templates is free forever. A Pro tier with custom templates is planned.

What if I don't have text selected?

The keyboard shortcut gracefully does nothing. Just select text first.
