# Prompt Clip — Chrome Web Store Listing

## Extension Name
**Prompt Clip — AI Prompt Builder**

## Short Description (132 chars max)
Turn selected text into ready-to-paste AI prompts with one click. Adds source context, smart formatting, and 5 templates. Free, private, no accounts.

## Long Description

### Stop formatting AI prompts by hand

You find an interesting paragraph, code snippet, or data table online. You want to ask an AI about it. Right now that means:

1. Copy the text
2. Switch to ChatGPT, Claude, or Gemini
3. Manually type "Here's some text: [paste] Can you summarize this?"
4. Repeat. Every. Single. Time.

**Prompt Clip eliminates the busywork.** Select text on any webpage, right-click → "Send to AI", pick a template, and a perfectly formatted prompt is on your clipboard — complete with page title, source URL, date, and the right instruction.

### How it works

1. **Select text** on any webpage
2. **Right-click** → "Send to AI" → choose a template (or press Ctrl+Shift+P for your last-used one)
3. **Paste** into any AI chat — the prompt is ready to send

### 5 Built-in Templates

- **Summarize** — "Please summarize the above in 3 bullet points."
- **Explain** — "Please explain the above in simple terms."
- **Translate** — "Please translate the above into clear, natural English."
- **Find Action Items** — "Please extract all action items from the above."
- **Code Review** — "Please review the above code for bugs, readability, and best practices."

### Smart Formatting

- Code snippets (detected automatically) get wrapped in markdown code fences
- Prose gets wrapped in blockquotes
- Every prompt includes source context: page title, URL, and date

### Keyboard Shortcut

Press **Ctrl+Shift+P** (Cmd+Shift+P on Mac) to instantly trigger your last-used template. A green checkmark confirms the copy.

### 100% Private

- No accounts required
- No network requests
- No data collection
- No tracking
- Everything runs locally in your browser

### Permissions Explained

- **contextMenus** — Shows the "Send to AI" right-click menu
- **scripting** — Reads your text selection from the page
- **storage** — Saves your template preferences locally
- **activeTab** — Accesses the current page's title and URL for context

### Coming in Pro

- Unlimited custom templates
- Template categories and folders
- Export prompt history
- Priority support
