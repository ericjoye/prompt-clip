# Prompt Clip — Chrome Web Store Listing

## Extension Name

**Prompt Clip — Send Text to AI**

## Short Description (132 chars max)

Turn selected text into ready-to-paste AI prompts with one click. Works with ChatGPT, Claude, and Gemini.

## Long Description

Prompt Clip is a Chrome extension that turns selected text into ready-to-paste AI prompts with one click. Select any text on a webpage, right-click, and choose from 5 built-in prompt templates — Summarize, Explain, Translate, Find Action Items, or Code Review. The formatted prompt is copied to your clipboard with automatic context framing, ready to paste into ChatGPT, Claude, or Gemini.

**CONTEXT-AWARE PROMPTS**
Every prompt includes automatic context: page title, source URL, current date, and your selected text. This means when you paste into an AI tool, it already knows where the content came from and what page you're referencing — no manual context entry needed.

**5 BUILT-IN TEMPLATES**
- **Summarize** — "Summarize the following content in 3 bullet points..."
- **Explain** — "Explain this to me like I'm a beginner..."
- **Translate** — "Translate this to [language]..."
- **Find Action Items** — "Extract all action items and deadlines..."
- **Code Review** — "Review this code for bugs, style, and best practices..."

**KEYBOARD SHORTCUT**
Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) with text selected to trigger the last-used template instantly. No right-click needed for repeat use.

**PRIVACY-FIRST**
Prompt Clip runs entirely in your browser. No data collection, no accounts, no tracking, no network calls. Your text stays on your device until you choose to paste it into an AI tool.

## Key Features

- **Right-click context menu** — "Send to AI" submenu with 5 templates
- **Auto-context framing** — includes page title, URL, date, and selected text
- **Keyboard shortcut** — `Ctrl+Shift+P` triggers last-used template
- **One-click clipboard copy** — no accounts, no network calls
- **Options page** — toggle templates on/off, choose default
- **Zero dependencies** — pure vanilla JS, no build step
- **Privacy-first** — no data collection, no tracking, no accounts
- **Manifest V3** — fully compatible with latest Chrome standard

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `prompt-clip/` folder
5. The extension icon appears in your toolbar

## Requirements

- Chrome 88+ (Manifest V3 support)
- No additional dependencies
