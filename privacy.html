Privacy Policy

Last updated: June 20, 2026

Overview

Prompt Clip ("we", "our", "us") is a Chrome extension that helps you turn selected text into AI prompts. We are committed to your privacy.

Data Collection

We do not collect any personal data. Prompt Clip operates entirely locally in your browser. No information is transmitted to external servers.

Data Storage

The only data stored by Prompt Clip is:

  • Your preferences (enabled templates, default template, keyboard shortcut) — stored locally via chrome.storage.local
  • No user accounts, no cloud sync, no analytics

Permissions

Prompt Clip requests the following permissions:

  • contextMenus — to add "Send to AI" to the right-click menu
  • scripting — to copy generated prompts to your clipboard
  • storage — to save your preferences locally
  • activeTab — to access the current page's title and URL for prompt context

Third Parties

Prompt Clip does not share data with any third parties. We do not use analytics, advertising, or tracking of any kind.

Changes to This Policy

If we update this policy, we will update the "Last updated" date and notify users through the extension's options page.

Contact

If you have questions about this privacy policy, please reach out through the Chrome Web Store listing.
